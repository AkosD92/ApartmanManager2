BigBuddy is forum website made with
	 - python
	 - flask
	 
Features:
	- log in/out system
	- new topics can be introduced with voting system
	- each comment can be flagged -> if limit reached, then the comment is removed
	- each removed comment causes a negative point to the user -> if the limit is reached,
		then the user gets a time-ban in the actual topic
	- each comment can be positively flagged -> the user can be rated in the actual topic
